## What I’m Building

### Parva

Parva is a pre-launch CRM for creative service businesses. It is designed to centralize client intake, project communication, and file delivery in one place.

The product is shaped by firsthand experience running a service business and seeing how quickly client details, notes, files, follow-ups, and communication can become scattered across tools.

Focus areas:
- Client intake workflows
- Project-based communication
- File and asset delivery
- Contact and project management
- AI-assisted product development

### Burlington Photo Booth Package-Fit Tool

A decision-support tool for event clients comparing photo booth packages. It helps clients understand which package structure makes sense based on event details, active booth hours, paused time, customization, and estimated cost.

This project came from a real commerce problem: clients were often not stuck on price, but on uncertainty around what they actually needed.

### rubiMESSAGE

At Rubicon, I helped take rubiMESSAGE from problem to shipped product. It is an SMS workflow tool for public safety teams that helps non-emergency calltakers send citizens the correct online report link while clearing phone lines faster.

This work strengthened how I think about product feedback loops, workflow clarity, and building tools for high-pressure operational environments.
